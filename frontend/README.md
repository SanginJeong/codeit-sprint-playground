# Codeit-Sprint-Playground

## 프로젝트 기간
2025-07-03 ~

## 프로젝트 목적
스프린트 수료 후에 더 강해진 나를 위해서 기록하기 위한 공간제작

## 기능
- 회원가입
- 로그인
- 파트1, 파트2, 파트3, 파트4
- 파트별 멤버, 기간 볼수 있다.
- 기간을 클릭하면 등록된 글들이 나온다.
- 글들을 클릭하면 해당 글 페이지로 이동한다.
- 유저의 편의성을 위해 사이드바에서 이전, 홈으로 이동 기능을 제공한다.
